 # Jarvis AI 🤖

Jarvis AI is a voice-controlled virtual assistant built with Python and inspired by Iron Man's J.A.R.V.I.S. It wakes up when you say **"Jarvis"** and can perform everyday tasks like opening websites, playing music, and answering questions using AI.

## ✨ Features
- 🎙️ Wake word activation ("Jarvis")
- 🌐 Open websites with voice commands:
  - Google
  - YouTube
  - Facebook
  - LinkedIn
  - GitHub
  - Cricbuzz
- 🎵 Play songs from a custom music library
- 🤖 Get answers using AI integration
- 🗣️ Text-to-speech responses
- 🎤 Speech recognition through microphone input

## 🛠️ Built With
- Python
- SpeechRecognition
- pyttsx4
- Webbrowser
- Google Speech Recognition API
- Gemini AI API

## 🚀 Why I Built This
I always wanted to create my own version of Iron Man's J.A.R.V.I.S., so I built this project to learn about voice recognition, AI integration, and automation using Python.
